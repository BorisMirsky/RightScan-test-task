

Задача №4.

Дано:
Таблица tasks:
| id | status | updated_at |

Задача:
Написать SQL-запрос:
- найти все задачи в статусе IN_PROGRESS
- которые не обновлялись более 10 минут


----------------------------------------------------------------------
----------------------------------------------------------------------


Пусть у нас СУБД SQL MS Server. Предполагаем, что в таблице данные трёх типов:
- id - int
- status - varchar(10)
- updated_at - Datetime

Из условия непонятно, что именно нужно:
- все записи, отвечающие условию 
- только последнее текущее значение. 

Поэтому приводится два решения.

Решение 1 - таблица хранит только текущее состояние задачи, т.е. на каждую задачу (на каждый id) есть только одна запись.

SELECT id 
FROM tasks 
WHERE status = 'IN_PROGRESS'

SELECT id 
FROM tasks 
WHERE updated_at < DATEADD(minute, -10, GETDATE())


----------------------------------------------------------------------

Решение 2 - таблица хранит историю изменений статусов, т.е. на каждую задачу (на каждый id) возможно более одной записи.

Для каждой задачи (для каждого id) берём последнюю по времени обновления запись со статусом == "IN_PROGRESS"

WITH LastInProgress AS (
    SELECT 
        id,
        status,
        updated_at,
        ROW_NUMBER() OVER (PARTITION BY id ORDER BY updated_at DESC) AS rn
    FROM tasks
    WHERE status = 'IN_PROGRESS'
)
SELECT id, status, updated_at
FROM LastInProgress
WHERE rn = 1;



Задачи с любым статусом (все записи таблицы), которые не обновлялись более 10 минут.

WITH LastTask AS (
    SELECT 
        id,
        status,
        updated_at,
        ROW_NUMBER() OVER (PARTITION BY id ORDER BY updated_at DESC) AS rn
    FROM tasks
)
SELECT id, status, updated_at
FROM LastTask
WHERE rn = 1
  AND DATEDIFF(MINUTE, updated_at, GETDATE()) > 10;
